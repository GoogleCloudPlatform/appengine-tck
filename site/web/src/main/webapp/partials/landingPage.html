Google App Engine TCK

Provides Technology Compatibility Kit (TCK) for Google App Engine (GAE) API

  • Reports
  • Coverage
  • View On GitHub

Goals

  • Comprehensive GAE API test coverage
  • GAE API usage examples
  • Common GAE API test environment

Can be used by external vendors implementing the GAE API - such as CapeDwarf and AppScale, or developers writing frameworks on top of GAE API, or simply using GAE API
